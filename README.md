## <b>Description</b>
Stima parser extracts interruption data from  scheduled power interruptions pdf documents with regex pattern-matching for the following fields:

1. Region
2. County
3. Area
4. Date
5. Time
6. Mtaa

## <b>Prerequisites</b>
- Python3
